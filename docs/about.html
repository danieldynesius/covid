About the Project

WARNING: This project is still in the Early Beta Stage. Focus currently is to Add Data Sources & functionality, Data Quality Control is Later Stage.

This website is hobby project to visualize Covid-19 Wastewater Data collected by multiple countries authorities or universities to aggregate and present the data in one place.

Feel free to contact me shareing datalinks to countries not yet included or any suggestions and ideas.
